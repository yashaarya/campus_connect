# Campus Connect

Campus Connect is a simple and intuitive web platform designed to help students quickly discover nearby sports facilities, including gyms, swimming pools, turfs, grounds, courts, and sports goods shops around their college.

---

## 🚀 Features

* Explore nearby sports places with clean UI
* View gyms, turfs, swimming pools, grounds, courts, and shops
* Minimal and fast-loading interface
* Easy navigation for students



## 💻 Technologies Used

* **HTML5** for structure
* **CSS3** for styling
* **JavaScript** (optional) for interactivity
* **Google Fonts** for typography



## 🛠️ How to Run Locally

1. Download or clone the repository.
2. Open the project folder.
3. Launch `homepage.html` in any modern browser.
